## 📌 О проекте

**Loompage** — это твоя личная страница для управления задачами, заметками и ссылками. Всё в одном месте, никакой регистрации, только ты и твой порядок.

### 🧵 Почему Loompage?

**Loom** (ткацкий станок) + **Page** (страница) = место, где ты **сплетаешь** разрозненные задачи, идеи и ресурсы в единую структуру. Как на ткацком станке из нитей создаётся полотно, так и здесь из хаоса рождается порядок.

### Что внутри?

- **📋 Канбан-доска** — визуальное управление задачами с колонками: To Do, В работе, На проверке, Готово, Архив
- **🔗 Ссылки** — хранилище полезных URL-адресов с категориями и древовидной структурой
- **📝 Заметки** — быстрые текстовые записи для идей, планов на дейлики и важных мыслей
- **💬 Отзывы** — форма для обратной связи с отправкой в Google Таблицу

Все данные сохраняются в локальном хранилище твоего браузера (localStorage) и не требуют регистрации или интернета после первой загрузки.

## ✨ Возможности

### 📋 Канбан-доска
- **Drag & Drop** — перетаскивай задачи между колонками
- **Фильтрация** — просматривай задачи по статусу
- **Поиск** — быстрый поиск по названию и описанию
- **Приоритеты** — высокая, средняя, низкая важность
- **Дедлайны** — устанавливай сроки выполнения
- **Категории** — группируй задачи по темам (Работа, Личное, Учеба, Здоровье)
- **Ссылки** — прикрепляй URL к задачам
- **Статистика** — общее количество, в работе, выполнено, архив
- **Экспорт в Excel** — выгружай все задачи в таблицу

### 🔗 Ссылки
- **Древовидная структура** — создавай вложенные категории
- **Drag & Drop** — перетаскивай категории для изменения порядка
- **Карточки ссылок** — название, URL, описание
- **Фильтрация по категориям** — просматривай ссылки в выбранной категории
- **Счётчик ссылок** — видишь количество ссылок в категории
- **Экспорт в Excel** — выгружай ссылки с категориями

### 📝 Заметки
- **Карточки заметок** — заголовок и текстовое содержимое
- **Дата создания** — автоматическая метка времени
- **Экспорт в Excel** — выгружай все заметки

### 💬 Отзывы
- **Форма обратной связи** — имя, email, текст отзыва, рекомендация, оценка звёздами
- **Отправка в Google Таблицу** — все отзывы сохраняются в облаке
- **Автоматическая дата** — время отправки фиксируется автоматически

### 🔧 Дополнительные функции
- **Бэкап данных** — экспорт/импорт всех данных в JSON
- **Тёмная тема** — комфортная работа в тёмное время суток
- **Адаптивный дизайн** — корректно отображается на всех устройствах
- **Анимированные иконки** — плавные переходы и приятные визуальные эффекты
- **Меню с дополнительными функциями** — всё необходимое в одном месте
